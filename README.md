# pr-url-scanner
This service scans PRs for URLs, verifies whether they are reachable, and reports the results in a comment.

## To build:
```
go get
go build -o main
```

## To run:
```
GITHUB_TOKEN=<your-token> ./main
```

## Dependencies:
mvdan.cc/xurls
